Politique de Confidentialité

Dernière mise à jour : 14 avril 2026

1. Collecte des données

Notre application utilise les API de TikTok pour accéder aux données publiques des vidéos (statistiques, légendes, métadonnées). Nous ne collectons aucune donnée personnelle de l'utilisateur sans son consentement explicite.

2. Utilisation des données

Les données récupérées via TikTok sont traitées exclusivement par notre instance automatisée sur n8n. Ces données sont utilisées uniquement à des fins d'analyse statistique et technique pour le compte de l'utilisateur.

3. Partage et Stockage

Nous ne vendons ni ne partageons vos données à des tiers. Les analyses peuvent être temporairement stockées sur vos propres services connectés (ex: Google Sheets, base de données privée) selon votre configuration n8n.

4. Services Tiers

En utilisant cette application, vous acceptez d'être lié par les Conditions d'Utilisation de TikTok.
